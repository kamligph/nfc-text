# Setup
## 1. Clone repository
```
git clone git@github.com:kamligph/nfc-text.git
```

## 2. Go to your directory
```
cd nfc-text
```

## 3. Install Libraries
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.
```
pip install -r requirements.txt
```

## 4. Run command
```
python run.py 巨大モウジャ
```
where `巨大モウジャ` is the malformed text
